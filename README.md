# Fasyl RAG Project

A simple Retrieval-Augmented Generation (RAG) chatbot built using Python and Ollama.

## Models Used

- Embedding Model: BGE Base v1.5
- Language Model: Llama 3.2 1B Instruct

## Technologies

- Python
- Ollama
- Vector Embeddings
- Cosine Similarity
- Retrieval-Augmented Generation (RAG)

## Run the Project

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install ollama
python demo.py
```
